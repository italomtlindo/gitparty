O Daily Check é um site desenvolvido para organizar tarefas e consultar a temperatura de uma cidade em tempo real. O sistema possui uma interface moderna com páginas de Home, Temperatura e Tarefas. As tarefas são exibidas em cards e armazenadas em banco de dados utilizando Prisma, unindo praticidade, organização e integração entre front-end e back-end.

Descrição do projeto
O projeto é um sistema de gestão de eventos desenvolvido para controlar participantes, eventos e inscrições de forma automatizada. O sistema resolve problemas como inscrições duplicadas, limite de vagas e gerenciamento de cancelamentos.

Telas e elementos
O sistema possui páginas de Usuários, Eventos e Inscrições. É possível cadastrar eventos, realizar inscrições e acompanhar o status das vagas e lista de espera.

Regras de negócio
O usuário não pode se inscrever duas vezes no mesmo evento. Quando as vagas acabam, novas inscrições entram na lista de espera. Cancelamentos só podem ser feitos até 24 horas antes do evento, e vagas liberadas promovem automaticamente participantes da lista de espera.

Estilo e cores
Interface moderna, organizada e responsiva, com navbar escura, fundo claro e formulários simples.

Público-alvo
Empresas e organizadores de palestras, workshops e treinamentos.

Referências
Inspirado em Eventbrite e Sympla.
